## 🏃 Running Locally

No build tools or dependencies required — it's plain HTML, CSS & JS.

1. **Clone the repository**
   ```bash
   git clone https://github.com/aish2004-cloud/personal-portfolio.git
   cd personal-portfolio
   ```

2. **Open in browser**

   Simply open `index.html` in your browser:
   ```bash
   open index.html
   # or on Windows
   start index.html
   ```

   Or use the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code for hot reload.

## 🌍 Deployment

This portfolio is deployed on **Vercel** for free hosting with automatic HTTPS.

To deploy your own version:

1. Push the project to GitHub
2. Go to [vercel.com](https://vercel.com/) and import the repository
3. Click **Deploy** — no configuration needed for a static site ✅
